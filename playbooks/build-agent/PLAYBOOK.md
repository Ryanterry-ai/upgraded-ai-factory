# Build Agent Playbook

## Overview
Step-by-step playbook for building a custom AI agent with specific capabilities, knowledge base, and integrations.

## When to Use
- User needs a custom AI chatbot
- User requires an agent with specific knowledge
- User needs an agent with API integrations

## Prerequisites
- Agent purpose and capabilities
- Knowledge base content
- Integration requirements
- UI requirements (if applicable)

## Steps

### Step 1: Requirements Gathering
**Agent**: PM Agent
**Action**: Analyze agent requirements
**Output**: Agent requirements document

### Step 2: Capability Design
**Agent**: Product Strategist Agent
**Action**: Design agent capabilities and personality
**Output**: Capability specification

### Step 3: Knowledge Base Design
**Agent**: Research Agents
**Action**: Design knowledge base structure
**Output**: Knowledge base schema

### Step 4: Architecture Design
**Agent**: Backend Architect Agent
**Action**: Design agent architecture
**Output**: Architecture documentation

### Step 5: Blueprint Generation
**Agent**: Blueprint Generator Agent
**Action**: Generate agent Blueprint
**Output**: `blueprint.yaml`

### Step 6: Blueprint Review
**Agent**: Coordinator Agent (Gate)
**Action**: Validate Blueprint completeness
**Output**: Approval or rework request

### Step 7: Agent Core Implementation
**Agent**: Backend Engineer Agent
**Action**: Implement agent core logic
**Output**: Agent core code

### Step 8: Knowledge Base Implementation
**Agent**: Backend Engineer Agent
**Action**: Implement knowledge base
**Output**: Knowledge base code

### Step 9: Integration Implementation
**Agent**: Backend Engineer Agent
**Action**: Implement API integrations
**Output**: Integration code

### Step 10: UI Implementation (if applicable)
**Agent**: Frontend Engineer Agent
**Action**: Build agent chat UI
**Output**: Chat interface components

### Step 11: Build & Test
**Agent**: Backend Architect + QA Engineer
**Action**: Build and test agent
**Output**: Build success, test results

### Step 12: Final Review
**Agent**: Coordinator Agent (Gate)
**Action**: Final quality gate
**Output**: Approval for deployment

### Step 13: Deployment
**Agent**: Deployment Agent
**Action**: Prepare deployment
**Output**: Deployment configuration

## Quality Checkpoints
- Step 6: Blueprint validation
- Step 11: Build and test
- Step 12: Final review

## Estimated Time
- 2-4 hours for simple agent
- 4-8 hours for complex agent
